# Sources and Links - Summer Springboard UC Berkeley 2026

## How to Use This Document

This document maps every topic in the knowledge base to its original source materials. When the chatbot provides an answer, it should embed the relevant links so users can reference the original documents.

**Format for Chatbot Responses:**
```
[Answer to question]

📚 **Source:** [Link description](actual URL)
```

## Important Notes for Chatbot Implementation

### When Portal Links Are Unavailable
If a user cannot access Google Sites links (due to permissions, network issues, etc.), the chatbot should:

1. Provide the information from the knowledge base
